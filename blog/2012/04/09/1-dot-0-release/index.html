OSXDev 버전 1.0 릴리즈

Apr 9th, 2012

안녕하세요.

드디어 잉여력을 발휘하며 개발해온 OSXDev가 나름, 스스로 1.0 버전을 달아도 괜찮다라고 판단하여 1.0 버전을 릴리즈하였습니다.

현재까지 구현한 주요 기능과 추후 구현할 기능은 다음과 같습니다.

버전 1.0 구현 목록

  • 포럼 리스트
  • 활성화 토픽 리스트
  • 포럼별 토픽 리스트 (포럼 정보)
  • 글타래 보여주기 (글타래 정보)
  • 로그인 (자동 로그인 포함)
  • 토픽 남기기
  • 댓글 남기기
  • 토픽 페이지 넘기기, 해당 페이지 이동
  • 댓글 페이지 넘기기, 해당 페이지 이동
  • 로그인 관련 리셋을 위한 세팅 번들 추가
  • 링크 클릭시 내부 브라우저로 보여주기 (사파리로 열기 포함)

추후 구현 목록

  • 토픽, 댓글 검색 기능
  • 토픽, 댓글 남기기에 사진 첨부하기 기능
  • 토픽, 댓글 세부 정보 보여주기 (조회수, 최근 등록 시간 등)
  • 토픽, 댓글 신고하기 기능
  • 사용자 프로필 사진 보여주기

모쪼록, 잉여력이 남는대로 틈틈히 작업하겠습니다. 감사합니다!
